## Read Me

作者：汤麻酱

版本：0.0.0

这是一个矩阵运算库，主体数据结构使用了CSDN用户[shuoyueqishilove](https://blog.csdn.net/shuoyueqishilove)的[博文](https://blog.csdn.net/shuoyueqishilove/article/details/80427501)中的矩阵，在其对矩阵的基本运算基础上添加了一些较为高级的算法。

包括：

> LUP分解
>
> 高斯消元法解线性方程组
>
> 幂法
>
> 反幂法
>
> 谱半径
>
> 1、2、无穷范数
>
> 1、2、无穷条件数
>
> Jacobi迭代法解线性方程组
>
> Gauss-Seidel迭代法解线性方程组

待续
